# Variables and Constant

***Variables***

A name givin to a storage location in memory

***Constant*** 

An immutable value

***Declaring Variables***

````csharp
int number;
int number = 1;
cons float Pi = 3.14f; 
````


***Naming Conventions***

- Camel Case -> ***f***irst***N***ame
- Pascal Case -> ***F***irst***N***ame
- HungarianNotation -> ***str***FirstName
